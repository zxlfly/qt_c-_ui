# 环形管旋转切换 使用说明

## 概述

环形管旋转切换示例展示了 `CircleDisplayWidget` 的旋转偏移能力。大管和小管作为一个整体旋转，顶部倒三角位置固定不动，每次旋转恰好让倒三角指向下一个小管。

演示页面包含 4 个独立实例，同时旋转。

适用场景：旋转阀门选择、工位切换、管道轮巡监控等。

---

## 交互说明

| 操作 | 效果 |
|------|------|
| 点击「全部下一个 ▶」 | 4 个控件同时顺时针旋转 30°，倒三角指向下一个小管 |
| 点击「◀ 全部上一个」 | 4 个控件同时逆时针旋转 30°，倒三角指向上一个小管 |
| 点击「全部复位」 | 4 个控件动画回归初始位置，倒三角重新指向 1 号管 |

- 旋转过程有平滑动画（400ms，缓入缓出曲线）
- 每个控件独立维护动画状态，动画进行中的控件跳过本次操作
- 每个控件下方独立显示当前倒三角指向的小管编号和名称

---

## 核心原理

### 旋转偏移属性

`CircleDisplayWidget` 提供了 `rotationOffset` 属性（Q_PROPERTY），控制大管+小管整体的旋转角度偏移：

```cpp
// 设置旋转偏移（度数）
widget->setRotationOffset(-30.0);   // 顺时针旋转 30°
widget->setRotationOffset(30.0);    // 逆时针旋转 30°
widget->setRotationOffset(0.0);     // 复位

// 读取当前偏移
double offset = widget->rotationOffset();
```

**关键特性**：
- 旋转偏移仅影响小圆、弧线、编号标签的位置
- **倒三角形始终固定在顶部**，不随旋转变化
- `hitTest()` 和 `itemCenter()` 同样受旋转偏移影响，点击检测和坐标定位始终正确

### 动画驱动

由于 `rotationOffset` 是 Q_PROPERTY，可以直接用 `QPropertyAnimation` 驱动实现平滑旋转动画：

```cpp
QPropertyAnimation *anim = new QPropertyAnimation(widget, "rotationOffset");
anim->setDuration(400);                          // 400ms
anim->setEasingCurve(QEasingCurve::InOutCubic);  // 缓入缓出
anim->setStartValue(widget->rotationOffset());
anim->setEndValue(widget->rotationOffset() - 30.0);  // 下一个小管
anim->start();
```

## 快速开始

### 最简用法

```cpp
#include "circledisplaywidget.h"

// 1. 创建控件
CircleDisplayWidget *widget = new CircleDisplayWidget(this);
widget->setAllTexts({"管1", "管2", "管3", "管4", "管5", "管6",
                     "管7", "管8", "管9", "管10", "管11", "管12"});
layout->addWidget(widget);

// 2. 直接设置旋转偏移（无动画，立即跳转）
widget->setRotationOffset(-60.0);   // 顺时针转2个位置
```

### 带动画的旋转

```cpp
#include "circledisplaywidget.h"
#include <QPropertyAnimation>

CircleDisplayWidget *widget = new CircleDisplayWidget(this);
layout->addWidget(widget);

// 旋转到下一个小管（带动画）
QPropertyAnimation *anim = new QPropertyAnimation(widget, "rotationOffset");
anim->setDuration(400);
anim->setEasingCurve(QEasingCurve::InOutCubic);
anim->setStartValue(widget->rotationOffset());
anim->setEndValue(widget->rotationOffset() - 30.0);  // 每个小管间隔 30°
anim->start();
```

### 完整示例：自定义旋转控制页

```cpp
#include "circledisplaywidget.h"
#include <QPropertyAnimation>
#include <QPushButton>
#include <QVBoxLayout>
#include <QHBoxLayout>
#include <QLabel>

class MyRotatePage : public QWidget
{
    Q_OBJECT
public:
    explicit MyRotatePage(QWidget *parent = nullptr) : QWidget(parent)
    {
        auto *layout = new QVBoxLayout(this);

        // 创建环形管控件
        m_circle = new CircleDisplayWidget(this);
        m_circle->setAllTexts({"原料A", "原料B", "原料C", "原料D",
                                "催化剂", "溶剂", "添加剂", "缓冲剂",
                                "稳定剂", "助剂", "混合物", "产物"});
        layout->addWidget(m_circle, 1);

        // 信息标签
        m_label = new QLabel("当前指向：1号管", this);
        layout->addWidget(m_label);

        // 按钮
        auto *btnLayout = new QHBoxLayout();
        auto *btnPrev = new QPushButton("◀ 上一个", this);
        auto *btnNext = new QPushButton("下一个 ▶", this);
        btnLayout->addWidget(btnPrev);
        btnLayout->addWidget(btnNext);
        layout->addLayout(btnLayout);

        // 创建动画
        m_anim = new QPropertyAnimation(m_circle, "rotationOffset");
        m_anim->setDuration(400);
        m_anim->setEasingCurve(QEasingCurve::InOutCubic);

        // 连接按钮
        connect(btnNext, &QPushButton::clicked, this, &MyRotatePage::rotateNext);
        connect(btnPrev, &QPushButton::clicked, this, &MyRotatePage::rotatePrev);
        connect(m_anim, &QPropertyAnimation::finished, this, [this]() {
            m_animating = false;
            m_label->setText(QString("当前指向：%1号管 — %2")
                .arg(m_step + 1).arg(m_circle->itemText(m_step)));
        });
    }

private slots:
    void rotateNext()
    {
        if (m_animating) return;
        m_animating = true;
        m_step = (m_step + 1) % 12;

        m_anim->setStartValue(m_circle->rotationOffset());
        m_anim->setEndValue(m_circle->rotationOffset() - 30.0);
        m_anim->start();
    }

    void rotatePrev()
    {
        if (m_animating) return;
        m_animating = true;
        m_step = (m_step + 11) % 12;

        m_anim->setStartValue(m_circle->rotationOffset());
        m_anim->setEndValue(m_circle->rotationOffset() + 30.0);
        m_anim->start();
    }

private:
    CircleDisplayWidget *m_circle;
    QPropertyAnimation *m_anim;
    QLabel *m_label;
    int m_step = 0;
    bool m_animating = false;
};
```

## 相关 API

### CircleDisplayWidget 旋转相关

| 方法 / 信号 | 说明 |
|-------------|------|
| `setRotationOffset(double degrees)` | 设置旋转偏移角度（度数），正值逆时针，负值顺时针 |
| `rotationOffset() const` | 获取当前旋转偏移 |
| `rotationOffsetChanged(double)` | 旋转偏移变化时发射的信号 |
| `itemText(int index) const` | 获取第 index 个小管的文字内容 |

### 旋转方向约定

| 偏移值变化 | 视觉效果 |
|-----------|---------|
| offset 减少（如 -30°） | 小管整体**顺时针**旋转，倒三角指向**下一个**小管 |
| offset 增加（如 +30°） | 小管整体**逆时针**旋转，倒三角指向**上一个**小管 |

| 步数 (m_currentStep) | 倒三角指向 | rotationOffset |
|---------------------|-----------|----------------|
| 0 | 1号管 | 0° |
| 1 | 2号管 | -30° |
| 2 | 3号管 | -60° |
| 3 | 4号管 | -90° |
| ... | ... | ... |
| 11 | 12号管 | -330° |

## 注意事项

1. **动画互斥**：动画未结束时不要重复启动，否则会覆盖前一个动画的终点值
2. **offset 无回绕**：`rotationOffset` 不会自动归零（如 -360° → 0°），长时间反复旋转可能产生很大的绝对值，但不影响绘制精度
3. **倒三角不旋转**：`drawHandle()` 不使用 `m_rotationOffset`，倒三角始终在顶部
4. **hitTest 自动适配**：旋转后点击小管的命中检测自动跟随旋转偏移，无需额外处理
5. **多控件动画 finished 只连接一次**：在构造函数中连接 `finished` 信号，避免每次旋转都重复连接导致回调多次触发

## 多控件使用

`CircleRotateDemoPage` 演示了 4 个独立实例同时旋转的方案：

### 关键设计

1. **每个控件独立状态**：`RotateState` 结构体保存 widget/anim/step/animating，互不干扰
2. **独立动画对象**：每个控件有自己的 `QPropertyAnimation`，可以同时运行
3. **动画 finished 只连接一次**：在构造函数中连接，避免重复连接

```cpp
for (int i = 0; i < 4; ++i) {
    // 创建控件和动画...

    // finished 信号只连接一次
    connect(anim, &QPropertyAnimation::finished, this, [this, i]() {
        if (i < m_circles.size()) {
            m_circles[i].animating = false;
            updateInfoLabel(i);
        }
    });
}
```

### 按钮操作全部控件

```cpp
void onRotateForward() {
    for (RotateState &rs : m_circles) {
        if (rs.animating) continue;  // 跳过正在动画的
        rs.animating = true;
        rs.anim->setStartValue(rs.widget->rotationOffset());
        rs.anim->setEndValue(rs.widget->rotationOffset() - 30.0);
        rs.anim->start();
        rs.step = (rs.step + 1) % 12;
    }
}
```

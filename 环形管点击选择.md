# 环形管点击选择

基于 `CircleDisplayWidget` 的点击选择功能，点击小管弹出选项列表，选择后更新管内文字。

---

## 交互说明

| 操作 | 效果 |
|------|------|
| 点击小管 | 在小管旁弹出选项列表 |
| 点击列表项 | 对应小管文字替换为选中项 |
| 点击列表外区域 | 关闭弹出列表 |
| 取消勾选「开启点击选择」 | 禁用点击选择功能，点击小管无反应 |
| 重新勾选 | 恢复点击选择功能 |

---

## 核心 API

### CircleDisplayWidget 相关接口

| 方法/信号 | 说明 |
|-----------|------|
| `setSelectEnabled(bool enabled)` | 是否开启点击选择（默认 `false`） |
| `selectEnabled() const` | 查询选择功能状态 |
| `setItemText(int index, const QString &text)` | 设置第 index 个小管的文字 |
| `itemText(int index) const` | 获取第 index 个小管的文字 |
| `itemCenter(int index) const` | 获取第 index 个小圆的中心像素坐标（用于弹出菜单定位） |
| `itemClicked(int index)` | 信号，点击小管时发射，-1 表示空白区域 |

## 快速开始

### 1. 最简用法

```cpp
#include "circledisplaywidget.h"

// 创建控件
CircleDisplayWidget *circle = new CircleDisplayWidget(this);
layout->addWidget(circle);

// 开启点击选择
circle->setSelectEnabled(true);

// 监听点击
connect(circle, &CircleDisplayWidget::itemClicked, this, [circle](int index) {
    if (index >= 0) {
        qDebug() << "点击了" << index << "号管";
    }
});
```

### 2. 配合弹出列表

```cpp
#include "circledisplaywidget.h"
#include <QListWidget>
#include <QFrame>

// 创建控件并开启选择
CircleDisplayWidget *circle = new CircleDisplayWidget(this);
circle->setSelectEnabled(true);

// 创建弹出框
QFrame *popup = new QFrame(this);
popup->setFrameShape(QFrame::StyledPanel);
popup->hide();

QVBoxLayout *popupLayout = new QVBoxLayout(popup);
QListWidget *listWidget = new QListWidget(popup);
listWidget->addItems({"选项A", "选项B", "选项C"});
popupLayout->addWidget(listWidget);

int currentIdx = -1;  // 保存当前点击的小管索引

// 点击小管 → 弹出列表
connect(circle, &CircleDisplayWidget::itemClicked, this, [this, circle, popup, &currentIdx](int index) {
    if (index < 0) { popup->hide(); return; }
    currentIdx = index;

    // 定位弹出框到小管旁边
    QPointF center = circle->itemCenter(index);
    QPoint pos = circle->mapToParent(center.toPoint());
    pos += QPoint(30, -popup->height() / 2);
    popup->move(pos);
    popup->show();
    popup->raise();
});

// 选择列表项 → 更新小管文字
connect(listWidget, &QListWidget::itemClicked, this, [circle, popup, &currentIdx](QListWidgetItem *item) {
    if (currentIdx < 0 || !item) return;
    int idx = currentIdx;
    popup->hide();                           // 先关弹窗
    circle->setItemText(idx, item->text());   // 再更新文字
    circle->repaint();                        // 强制重绘
});

// 点击外部区域关闭弹窗（全局事件过滤器）
qApp->installEventFilter(this);  // 在类中重写 eventFilter
```

### 3. 事件过滤器示例

```cpp
bool MyPage::eventFilter(QObject *watched, QEvent *event)
{
    if (popup->isVisible() && event->type() == QEvent::MouseButtonPress) {
        auto *mouseEvent = static_cast<QMouseEvent *>(event);
        QPoint globalPos = mouseEvent->globalPosition().toPoint();
        QPoint popupLocal = popup->mapFromGlobal(globalPos);

        if (!popup->rect().contains(popupLocal)) {
            popup->hide();  // 点击在弹出框外部 → 关闭
        }
        // 关键：始终返回 false，不消费事件
    }
    return false;
}
```

## 实现要点

### 弹出列表定位

```cpp
QPointF center = circleWidget->itemCenter(index);     // 获取小圆中心
QPoint pos = circleWidget->mapToParent(center.toPoint()); // 转换到父控件坐标
pos += QPoint(30, -popup->height() / 2);               // 偏移到右侧
```

需要处理边界情况，确保弹出框不超出父控件：
```cpp
int maxX = width() - popup->width() - 4;
int maxY = height() - popup->height() - 4;
pos.setX(qBound(4, pos.x(), maxX));
pos.setY(qBound(4, pos.y(), maxY));
```

### 文字更新顺序

**先关闭弹窗，再更新文字**——避免弹窗遮挡重绘区域导致闪烁：

```cpp
int idx = currentIndex;
QString text = item->text();
hidePopup();                        // 先关
circleWidget->setItemText(idx, text); // 再更新
circleWidget->repaint();            // 强制重绘
```

### 事件过滤器注意事项

- 使用 `mapFromGlobal` + `rect().contains()` 判断点击是否在弹出框内
- **始终返回 `false`**，不消费事件，确保 QListWidget 的 `itemClicked` 信号正常发出
- 不要使用 `isAncestorOf(watched)` 判断，因为 QListWidget 内部的 viewport 子控件会导致误判

## 多控件使用

`CircleSelectDemoPage` 演示了 4 个 CircleDisplayWidget 实例共享一个弹出列表的方案：

### 关键设计

1. **共用弹出列表**：4 个控件共用同一个 `QFrame + QListWidget`，避免创建多个弹窗
2. **区分控件来源**：用 `m_currentCircle` 记录当前操作的是第几个控件（0~3），`m_currentIndex` 记录点击的小管索引（0~11）
3. **信号识别**：在 `itemClicked` 回调中用 `qobject_cast<CircleDisplayWidget *>(sender())` 找到信号来源控件

```cpp
// 连接每个控件的点击信号，用 lambda 捕获索引 i
for (int i = 0; i < 4; ++i) {
    connect(cw, &CircleDisplayWidget::itemClicked, this,
            [this, i](int index) { onCircleItemClicked(index); });
}

// 通过 sender() 找到是哪个控件发出的
void onCircleItemClicked(int index) {
    auto *sender = qobject_cast<CircleDisplayWidget *>(this->sender());
    // 遍历找到索引...
}
```

### 弹窗坐标映射

当控件在 GroupBox 等嵌套容器中时，弹窗定位需要多一层坐标转换：

```cpp
QPointF center = cw->itemCenter(itemIdx);
QPoint popupPos = cw->mapToParent(center.toPoint());
popupPos += QPoint(30, -popup->height() / 2);
// 关键：从 GroupBox 坐标映射到页面坐标
QPoint pagePos = cw->parentWidget()->mapToParent(popupPos);
```

## 注意事项

1. **事件过滤器始终返回 false**：不消费事件，确保 QListWidget 的 `itemClicked` 信号正常发出
2. **先关弹窗再更新文字**：避免弹窗遮挡重绘区域导致闪烁
3. **多控件需区分来源**：用 `sender()` 或 lambda 捕获索引识别是哪个控件发出的点击
4. **嵌套容器坐标映射**：控件在 GroupBox 中时，弹窗定位需从控件父层映射到页面层
